## 快速检索命令（PowerShell）

### 按关键词找会话（例如“改改”）
```powershell
rg -n --hidden "改改" C:\Users\MT\.codex\sessions C:\Users\MT\.codex\archived_sessions
```

### 按主题查（例如 docker / 云端 / ppt / 记忆）
```powershell
rg -n --hidden "docker|云端|ppt|记忆" C:\Users\MT\.codex\sessions C:\Users\MT\.codex\archived_sessions C:\Users\MT\.codex\.codex-global-state.json
```

### 查看最近会话文件
```powershell
Get-ChildItem -Force C:\Users\MT\.codex\sessions -Recurse |
  Sort-Object LastWriteTime -Descending |
  Select-Object -First 20 FullName, LastWriteTime, Length
```
