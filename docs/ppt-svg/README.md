# SVG 答辩页

本目录提供 4 页 SVG 版答辩页，面向答辩老师展示“数据报表怎么得出、怎么统计、最后说明什么”。

参考风格：

- `C:\Users\MT\Desktop\碳眸——校园中的低碳行为识别与分析.pptx`

主题特征：

- 主色：`#0060EB`、`#007EF9`
- 辅色：`#FFD671`、`#DDA301`
- 灰蓝：`#677684`
- 画布比例：`1920 x 1080`

文件说明：

- `slide-1-cover.svg`：封面与四类报表总览
- `slide-2-building-report.svg`：楼栋类报表怎么得出、怎么排序、怎么看
- `slide-3-trend-behavior.svg`：趋势图、专业得分图、行为影响图怎么统计
- `slide-4-closure.svg`：页面闭环与答辩收束结论

使用方式：

1. 可直接插入 PowerPoint / WPS。
2. 如果需要继续改字，建议先复制一份，再在矢量编辑器或 PPT 中解组微调。
