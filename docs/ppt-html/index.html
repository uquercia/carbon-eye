顶部指标卡片

数据表
能耗预测结果表提供各时间批次的全校用电、用水实际值和预测值；识别样例表提供训练后可展示的识别样例；行为影响规则表提供行为规则库数量；楼栋基础信息表提供楼栋覆盖数量。
计算方式
趋势指数由当前实际值相对历史平均值的偏离程度、最新预测值相对当前实际值的上行压力共同计算得到。识别样例数量来自识别样例表中的样例记录总数，行为规则库数量来自行为影响规则表，覆盖楼栋数来自楼栋基础信息表。
展示结论
这一组卡片先回答 3 个问题：系统覆盖多少楼、当前能不能演示识别、水电趋势是否活跃。

楼栋水电数据可视化预测概览

数据表
楼栋基础信息表提供楼栋名称、区域和专业映射；能耗预测结果表提供最新一批水电实际值、预测值和误差。
计算方式
先取时间批次最大的那一组数据作为当前状态；用电水平 = 本楼实际用电量 ÷ 全部楼栋最高实际用电量 × 100；用水水平 = 本楼实际用水量 ÷ 全部楼栋最高实际用水量 × 100；排序时优先按用电水平从高到低排列，再结合用水水平和综合分值。
展示结论
表格用脱敏进度条代替原始数值，方便答辩时直接说明 谁更高、差距多大、预测是否稳定。
页面区域：顶部概览卡片与楼栋预测概览表

part 02 / 能耗与行为关联图

楼栋关系气泡图展示不同建筑在校园空间中的能耗相对强度

农环大楼 生科大楼 应科大楼 诊断中心 人文大楼

报表样例图

气泡位置来自楼栋坐标，气泡大小来自楼栋实际用电相对值。

数据与计算说明

数据表
楼栋基础信息表提供楼栋名称、区域、专业映射和地图坐标；能耗预测结果表提供楼栋实际用电、实际用水和预测误差。
计算方式
先取最新时间批次的数据；气泡大小 = 基础尺寸 30 + 根号（本楼实际用电量 ÷ 全部楼栋最高实际用电量）× 46；颜色按用电占比划分为低强度、中强度和高强度。
展现数据
图中展示的是脱敏后的空间化能耗水平，教师可以看到高能耗楼栋是否在同一区域集中，也可以通过悬停查看区域、专业映射、用电水平、用水水平和预测状态。
分析结论
该图用于从校园空间视角定位重点楼栋，将“哪栋楼能耗高”与“楼栋位置和功能属性”联系起来。
数据口径：最新时间批次的楼栋记录

part 03 / 行为识别样例与低碳行为影响占比

行为类报表展示低碳行为识别样例，以及各行为对水电消耗的影响权重

上传校园图片后生成行为影响分析 行为名称、位置、置信度、用电影响、用水影响
低碳行为识别当前展示训练后样例数据

行为识别任务与结果分析

数据表
训练识别样例表提供演示样例；真实上传后使用上传图片表、识别任务表和识别结果表。
计算方式
样例数量 = 训练识别样例表中的记录总数；图片上传后，每条识别结果会保存置信度、预估用电影响量和预估用水影响量。
展示内容
该模块展示图片识别链路的输出结构，包括行为名称、发生位置、置信度和对用电、用水的估计影响。
  • 随手关灯关空调
  • 主动报修漏水漏电
  • 集中洗涤衣物
  • 缩短淋浴时间

低碳行为影响占比

数据表
行为影响规则表记录每类低碳行为名称、行为类别、说明文字、用电影响因子和用水影响因子。
计算方式
每类行为权重 = 用电影响因子的绝对值 + 用水影响因子的绝对值；占比 = 单个行为权重 ÷ 全部行为权重总和 × 100%。
展示内容
该报表展示哪些低碳行为对水电消耗影响更明显，用于说明宣传和干预优先级。
行为占比由行为影响规则表计算生成

part 05 / 趋势统计图

趋势统计图展示全校水电实际值与预测值走势，以及不同专业低碳行为得分

用电趋势实际/预测对比

数据表
能耗预测结果表。
计算方式
以时间批次为单位，把同一批次内所有楼栋的实际用电量相加，得到全校实际用电总量；再把同一批次内所有楼栋的预测用电量相加，得到全校预测用电总量。顶部用电趋势指数由当前实际用电总量相对历史平均用电总量的偏离程度、最新预测用电总量相对当前实际用电总量的上行压力共同计算得到。
展示内容
展示全校用电实际走势与预测走势的相对变化。

用水趋势实际/预测对比

数据表
能耗预测结果表。
计算方式
以时间批次为单位，把同一批次内所有楼栋的实际用水量相加，得到全校实际用水总量；再把同一批次内所有楼栋的预测用水量相加，得到全校预测用水总量。顶部用水趋势指数由当前实际用水总量相对历史平均用水总量的偏离程度、最新预测用水总量相对当前实际用水总量的上行压力共同计算得到。
展示内容
展示全校用水实际走势与预测走势的相对变化。

专业低碳行为得分

数据表
专业行为得分表。
计算方式
每个专业的综合得分由问卷中多项低碳行为得分汇总得到，页面直接展示各专业类别的综合平均分，分值范围为 0 到 5 分。
展示内容
展示不同专业群体低碳行为习惯的得分差异。
趋势数据由后端按时间批次聚合生成
